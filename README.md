# Nirmal Developers

A modern and responsive construction company website built using React and Tailwind CSS.

## Features

- Modern luxury UI design
- Fully responsive layout
- Home Page
- About Page
- Contact Page
- FAQ Section
- WhatsApp Integration
- Google Maps Integration
- Smooth Scroll Reveal Animations
- Construction Services Showcase
- Premium Tailwind CSS Styling

## Technologies Used

- React.js
- React Router DOM
- Tailwind CSS
- JavaScript (ES6+)
- HTML5
- CSS3

## Installation

Clone the repository:

```bash
git clone https://github.com/SagarKale18/Nirmal-Developers.git
```

Navigate to the project folder:

```bash
cd Nirmal-Developers
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Build for Production

```bash
npm run build
```
